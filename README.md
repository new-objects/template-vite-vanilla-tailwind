# Template: Vite + VanillaJS + TailwindCSS

## Usage

```bash
# Install dependencies
npm install

# Init husky and install peer dependencies
npm run prepare

# Run dev server
npm run dev

# Build for production
npm run build

# Format code (eslint and prettier)
npm run format
```
